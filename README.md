# Simple ATM simulator built with React.

## Technologies
- React.js
- JavaScript (Vanilla)
- SCSS

added LocalStorage functionality for storage balance data

## Features
- Account balance
- Cash withdrawal and deposit
- Simple Data storage
- Transaction history (#ToDo)
- PIN verification (#ToDo)

## Quick Start

```bash
npm install
npm start
```
